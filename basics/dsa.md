* **Data Structure** ‚Üí How data is stored and organized

## üì¶ Data Structures (DS)

Data Structures are different ways to store and organize data in memory.

### Common Data Structures

| Data Structure | Use Case                |
| -------------- | ----------------------- |
| Array          | List of elements        |
| String         | Text processing         |
| Object / Map   | Fast lookup             |
| Stack          | Undo, backtracking      |
| Queue          | Scheduling, async tasks |
| Linked List    | Dynamic memory          |
| Tree           | Hierarchical data       |
| Graph          | Networks, routes        |

### Example (JavaScript)

```js
const arr = [1, 2, 3, 4];
const map = new Map();
map.set('a', 1);
```

## ‚è±Ô∏è Time & Space Complexity

DSA focuses on how fast and memory-efficient your code is.

### Common Time Complexities

| Complexity | Meaning          |
| ---------- | ---------------- |
| O(1)       | Constant time    |
| O(n)       | Linear time      |
| O(n¬≤)      | Quadratic (slow) |
| O(log n)   | Very fast        |

### Example

```js
// O(n)
for (let i = 0; i < n; i++) {}

// O(n¬≤)
for (let i = 0; i < n; i++) {
  for (let j = 0; j < n; j++) {}
}
```
